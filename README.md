# System-Karyawan

Aplikasi monolith sederhana menggunakan CodeIgniter 4 dan SQL Server.

## Fitur
- CRUD Karyawan
- CRUD Departemen
- DataTables + AJAX
- JOIN Query
- SQL Server Database

## Teknologi
- PHP 8.5
- CodeIgniter 4
- SQL Server
- Bootstrap
- jQuery DataTables

## Menjalankan Project

```bash
composer install
php spark serve
```

## Endpoint
- /karyawan
- /departemen
