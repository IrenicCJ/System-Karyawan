# Sistem Karyawan

Aplikasi monolith sederhana menggunakan CodeIgniter 4 dan SQL Server.

## Fitur

- CRUD Data Karyawan
- Menggunakan AJAX & DataTables
- Menggunakan Query Join
- Database SQL Server
- Authentication Login

## Tech Stack

- CodeIgniter 4
- PHP 8
- SQL Server
- Bootstrap
- jQuery
- DataTables

## Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/USERNAME/System-Karyawan.git
```

### 2. Masuk Folder Project

```bash
cd System-Karyawan
```

### 3. Install Dependency

```bash
composer install
```

### 4. Setup Environment

Copy file:

```txt
env
```

menjadi:

```txt
.env
```

Lalu sesuaikan konfigurasi database SQL Server.

### 5. Jalankan Project

```bash
php spark serve
```

Project berjalan di:

```txt
http://localhost:8080
```
